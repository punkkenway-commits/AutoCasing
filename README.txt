  BASIRA — Autonomous Casing Architecture Generator
  Version 3.0
══════════════════════════════════════════════════════════════

HOW TO USE:
  1. Open "basira.html" in any modern browser
     (Chrome, Edge, Firefox, Safari)
  2. The app works OFFLINE — no internet required
  3. All libraries are bundled inside the single HTML file

BUNDLED LIBRARIES (no internet needed):
  • Plotly.js  2.27.0  — Charts & visualizations
  • Chart.js   4.4.0   — Design load charts
  • SheetJS    0.18.5  — Excel file import

FEATURES:
  • Casing grade selection (API 5CT / NACE MR0175 / ISO 15156)
  • Load calculations (Burst, Collapse, Axial) — StressCheck basis
  • PP/FG pressure profile (paste, CSV, LAS, Excel upload)
  • Mud Window chart (PP/FG/MW vs depth — SG units)
  • Design Load Envelope (per StressCheck)
  • Triaxial / Design Limits (Von Mises ellipse)
  • Cost estimation (AFE ±30%)
  • PDF / CSV / JSON / TXT reports

STANDARDS:
  API 5CT, API 5C3, NACE MR0175, ISO 15156, API RP 96,
  API TR 5C3, ISO 13680, StressCheck company standard

FILE SIZE: ~4.4 MB (all-in-one, no dependencies)
